# Phase 5 — PowerShell Automation Scripts

## Objective
Build PowerShell scripts to automate common Active Directory 
administration tasks, replacing manual processes with 
repeatable, logged automation.

## Scripts Repository

All scripts built during this phase are maintained in a 
dedicated repository:

### 🔗 [AD Automation Scripts](https://github.com/keey-keys/AD-Automation-Scripts)

| Script | Purpose |
|---|---|
| Find-InactiveAccounts.ps1 | Detect accounts inactive 90+ days |
| Unlock-UserAccount.ps1 | Unlock AD accounts with audit logging |
| Monitor-DiskSpace.ps1 | Alert when drives exceed 80% capacity |
| Get-UserAuditReport.ps1 | Export full user audit report to CSV |

## Key Concepts Learned
- PowerShell parameter binding
- Active Directory cmdlets
- Logging and audit trails
- Modular script design
- Running scripts via execution policy
